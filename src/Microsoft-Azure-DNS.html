cloud23 runs on Azure DNS

Het domein Cloud23.nl draait op Microsoft Azure DNS Services. Waarom? ik wilde graag de mogelijkheden van Azure DNS leren en bekijken. Via Microsoft Azure DNS heb je toegang tot de volgende opties.

De Microsoft Azure DNS maakt gebruik van een wereldwijd netwerk van Azure DNS servers in meer dan 54 Azure regios. Hierdoor is er altijd een DNS server dichbij om je DNS request te behandelen.

I

Azure DNS

Microsoft heeft al een aantal jaar een DNS optie binnen Azure. Binnen Azure DNS kunnen de DNS records voor je geregistreerde domeinen worden beheerd. Azure DNS onderscheid zich in de hoge beschikbaar van 99.99% uptime SLA verdeeld over de negenenvijftig Azure zones wereldwijd. 

my.cloud23.nl is een hobby project waar ik mijn ervaring m.b.t Microsoft producten laat zien.
